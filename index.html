About this test

        
      

      

        
What it does. This demo compares Backblaze B2 (S3-compatible) with AWS S3 by uploading and then downloading a set of objects with configurable size, count, and concurrency.

        
    
          
  • Reports throughput (MB/s) and latency percentiles (p50/p95/p99) per provider/op.
    • 
          
  • Uses AWS SDK v3; B2 via the S3-compatible endpoint you configure.
    • 
        

        
Not a formal benchmark. Results vary by instance size, region, network path, object size, and concurrency. For fair comparisons, run multiple rounds and keep provider regions/topology comparable.

        
Housekeeping. Test objects are written under a default prefix (e.g. perf-ui/).
